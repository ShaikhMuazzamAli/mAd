After you select Attendance from the Standard Profiles landing page, the Attendance Profiles list page opens. This page is a grid that displays the current Attendance Profiles and the attributes of each profile. To modify the attribute headings, click and select or clear the listed options.

  • To edit an Attendance Profile, select a profile from the list and click Edit.
  • To export the list page to an Excel .csv file, click Export , then select the .csv link at the bottom of the page.
